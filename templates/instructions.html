FaceSwap

1. What this app does?

This app takes two images as input and swap face of second image(Secondary image) with face of first image(main image).

2. How to use?

  • Main Image : This face will be in final result
  • Secondary Image : This image's body will be in final result
  • Both images should have human face
  • Images should not be too dark or too bright
  • Clear Button : Clears all the selected files
  • Swap Button : Swap the faces
